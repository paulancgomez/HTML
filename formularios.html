FORMULARIOS

  • Necesidad de interactuar con los usuarios.
  • Componente de una página web. <form> </form>
  • Existen controles para que el usuario proporcione información.
  • El procesamiento de esa información es en el servidor (más adelante).

Definir

  • DONDE se mandan los datos
  • COMO se mandan los datos

<form action="/donde" method="como">
</form>

Elementos de Entrada

  • Campo de texto
  • Casilla de verificación (Checkbox)
  • Botón de radio (radiobutton)
  • Cuadro de selección (select)
  • Botón de enviar (submit)
  • Botón de reinicio (Reset)
  • Selección de archivo (upload)
  • Campo de contraseña
  • etc.
  • etc (HTML5).
